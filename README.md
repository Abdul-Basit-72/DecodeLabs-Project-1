# DecodeLabs - Project 1: Responsive Frontend Interface 🚀

📝 Project Overview
This project is the first milestone of the DecodeLabs Internship (Batch 2026). The main objective was to build a **Mobile-First Responsive Layout** without relying on any external CSS frameworks (like Bootstrap or Tailwind). I have crafted a solid, adaptable, and modern frontend foundation using pure HTML, CSS, and Vanilla JavaScript.

## ✨ Key Features
- **Mobile-First Approach:** The layout is initially designed for mobile viewports and seamlessly scales to desktop screens using CSS Grid and Flexbox.
- **Semantic HTML5:** Utilized proper landmark elements (`<header>`, `<nav>`, `<main>`, `<article>`, `<footer>`) to ensure accessibility (WCAG compliance) and better SEO.
- **2025 UI/UX Aesthetics:** Strictly adhered to the DecodeLabs design guidelines, featuring a warm and grounded color palette ("Mocha Mousse", "Ethereal Blue", and "Moonlit Grey").
- **Premium CSS Upgrades:** Enhanced the UI with modern touches like glassmorphism, smooth lift-on-hover card animations, and soft shadows.
- **Interactive Navigation:** Implemented a fully functional mobile hamburger menu and dynamic active-link highlighting using Vanilla JavaScript.

## 🛠️ Tech Stack
- **HTML5** (Structure & Semantics)
- **CSS3** (Styling, Grid, Flexbox, Animations)
- **Vanilla JavaScript** (DOM Manipulation & Interactivity)
